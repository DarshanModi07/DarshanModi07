# ◈ Featured Project

## 🚀 Synapse

Enterprise workspace platform designed to streamline collaboration, communication, and project execution across organizations.

### Core Features

* Multi-tenant Workspace Architecture
* Role Based Access Control (RBAC)
* Department & Team Management
* Project & Task Tracking
* Real-time Notifications
* Profile Management
* Cloudinary Media Storage
* JWT Authentication
* PostgreSQL Relational Data Modeling

### Architecture

```text
Client
   │
   ▼
API Layer
   │
   ▼
Business Logic
   │
   ▼
Prisma ORM
   │
   ▼
PostgreSQL
```

### Tech Stack

```text
Node.js
Express.js
PostgreSQL
Prisma ORM
Socket.IO
Cloudinary
JWT
```
